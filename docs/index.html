asyncio-manager v1.0.0

Cliente AMI (Asterisk Manager Interface) asíncrono moderno para Python 3.10+.

Introducción

asyncio-manager es un reemplazo moderno de la librería Panoramisk (deprecada). Conecta con Asterisk vía el protocolo AMI usando async/await, sin dependencias externas, con type hints completos y manejo robusto de errores.

Async/await — Sin get_event_loop() deprecado
Cero deps — Solo librería estándar
Type hints — mypy --strict listo
Context manager — async with Manager(...) as m:
Reconexión — Backoff exponencial + jitter
EventList — Soporte explícito para listas de eventos

Instalación

pip install asyncio-manager

Con dependencias de desarrollo:

pip install -e ".[dev]"

Configuración requerida en /etc/asterisk/manager.conf:

[general]
enabled = yes
port = 5038
bindaddr = 0.0.0.0

[admin]
secret = password
read = all

import asyncio
from asyncio_manager import Manager

async def main():
    async with Manager(

        response = await manager.send_action({"Action": "Ping"})

asyncio.run(main())

Event Listener

Escucha todos los eventos de Asterisk y ejecuta callbacks asíncronos según el patrón registrado:

import asyncio
from asyncio_manager import Manager

async def main():
    async with Manager(
        host="127.0.0.1",
        username="admin",
        secret="password",
    ) as manager:
        await manager.connect()

        @manager.register_event("*")
        async def on_event(message):
            print(f"Evento: {message.event_type} - {message}")

        @manager.register_event("NewChannel")
        async def on_new_channel(message):
            print(f"Nueva llamada: {message.get('Channel')}")

        @manager.register_event("Hangup")
        async def on_hangup(message):
            print(f"Llamada finalizada: {message.get('Channel')}")

        try:
            while True:
                await asyncio.sleep(1)
        except KeyboardInterrupt:
            print("Cerrando...")

asyncio.run(main())

Originar llamadas

Usando el helper originate() del Manager, con soporte para llamadas asíncronas y concurrentes:

import asyncio
from asyncio_manager import Manager

async def originate_call(manager, phone_number):
    try:
        response = await manager.originate(
            channel=f"SIP/provider/{phone_number}",
            exten="100",
            context="from-internal",
            priority=1,
            caller_id="Test <1234>",
            timeout=30,
            async_=True,
        )
        if response.is_success:
            print(f"OK: {response.action_id}")
            return response.action_id
    except Exception as e:
        print(f"Error: {e}")
    return None

async def main():
    async with Manager(
        host="127.0.0.1",
        username="admin",
        secret="password",
        timeout=10.0,
    ) as manager:
        await manager.connect()

        numeros = ["5551234567", "5559876543"]
        tasks = [originate_call(manager, n) for n in numeros]
        results = await asyncio.gather(*tasks)
        print(f"Exitosas: {sum(1 for r in results if r)}/{len(results)}")

asyncio.run(main())

Servidor FastAGI

Servidor IVR asíncrono. A diferencia de Panoramisk, no usa el decorador asyncio.coroutine() (eliminado en Python 3.11):

import asyncio
from asyncio_manager import FastAGIServer, Request

async def ivr_handler(request: Request):
    channel = request.headers.get("agi_channel", "unknown")
    caller_id = request.headers.get("agi_callerid", "unknown")
    print(f"IVR: {caller_id} en {channel}")

    await request.answer()
    await request.say_digits("1")  # Presione 1
    await request.say_digits("2")  # Presione 2

    result = await request.get_data("welcome", 5000, 2)
    print(f"Usuario presionó: {result}")

    if result == "1":
        await request.exec_("Goto", "sales,s,1")
    elif result == "2":
        await request.exec_("Goto", "support,s,1")
    else:
        await request.hangup()

async def main():
    server = FastAGIServer()
    server.add_script("menu", ivr_handler)
    await server.start(host="0.0.0.0", port=4574)
    print("FastAGI escuchando en :4574")

    try:
        while True:
            await asyncio.sleep(1)
    except KeyboardInterrupt:
        await server.stop()

asyncio.run(main())

CallManager

Seguimiento de llamadas individuales con ciclo de vida completo:

import asyncio
from asyncio_manager import CallManager

async def main():
    call_mgr = CallManager(
        host="127.0.0.1",
        username="admin",
        secret="password",
    )

    async with call_mgr:
        call = await call_mgr.originate(
            channel="SIP/2000",
            exten="1000",
            context="default",
            priority=1,
        )
        print(f"Llamada originada: {call.id}")

        try:
            await call.wait_for_answer(timeout=30.0)
            print("Llamada respondida")

            await call.wait_for_hangup()
            print("Llamada finalizada")

        except asyncio.TimeoutError:
            print("Timeout - colgando...")
            await call.hangup()

asyncio.run(main())

EventList

Acciones que generan múltiples eventos (como QueueStatus) seguidas de un evento *Complete. Usa send_action_and_wait_all() para recolectarlos todos:

import asyncio
from asyncio_manager import Manager

async def main():
    async with Manager(
        host="127.0.0.1",
        username="admin",
        secret="password",
    ) as manager:
        await manager.connect()

        messages = await manager.send_action_and_wait_all({
            "Action": "QueueStatus",
            "Queue": "support",
        })

        print(f"Recibidos {len(messages)} mensajes:")
        for msg in messages:
            if msg.is_event:
                print(f"  {msg.event_type}: {dict(msg)}")

asyncio.run(main())

CLI (Interfaz de línea de comandos)

El paquete incluye un comando asyncio-manager para operaciones rápidas desde la terminal:

Enviar una acción

asyncio-manager --username admin --secret password action Ping

Originar llamada

asyncio-manager --username admin --secret password \
    originate SIP/100 200 --caller-id "Test <1234>"

Monitorear eventos

asyncio-manager --username admin --secret password \
    monitor --filter "NewChannel"

Ejecutar comando Asterisk

asyncio-manager --username admin --secret password \
    command "pjsip show endpoints"

FAQ / Troubleshooting

ConnectionError: No se puede conectar

Verifica que Asterisk esté corriendo (systemctl status asterisk) y el puerto AMI (5038) sea accesible (netstat -tlnp | grep 5038).

AuthenticationError: Credenciales inválidas

Revisa /etc/asterisk/manager.conf y asegúrate de que el usuario tenga permisos de lectura/escritura.

TimeoutError: La operación excedió el tiempo de espera

Aumenta los timeouts en la creación del Manager o al enviar la acción:

Manager(timeout=15.0, read_timeout=60.0, connect_timeout=30.0)

¿Cómo verifico que la librería funciona sin Asterisk real?
