asyncio-manager v1.0.0

Modern async Asterisk Manager Interface (AMI) client for Python 3.10+.

Introduction

asyncio-manager is a modern replacement for the deprecated Panoramisk library. It connects to Asterisk via the AMI protocol using async/await, with zero external dependencies, full type hints, and robust error handling.

Async/await — No deprecated get_event_loop()
Zero deps — Python stdlib only
Type hints — mypy --strict ready
Context manager — async with Manager(...) as m:
Reconnection — Exponential backoff + jitter
EventList — Explicit support for event lists

Installation

pip install asyncio-manager

With dev dependencies:

pip install -e ".[dev]"

[general]

port = 5038
bindaddr = 0.0.0.0

[admin]
secret = password
read = all
write = all

Quick Start

Basic connection

import asyncio
from asyncio_manager import Manager

async def main():
    async with Manager(
        host="127.0.0.1",
        username="admin",
        secret="password",
    ) as manager:
        await manager.connect()

        response = await manager.send_action({"Action": "Ping"})
        print(f"Ping success: {response.is_success}")

asyncio.run(main())

Event Listener

Listen to all Asterisk events with async callbacks:

import asyncio
from asyncio_manager import Manager

async def main():
    async with Manager(
        host="127.0.0.1",
        username="admin",
        secret="password",
    ) as manager:
        await manager.connect()

        @manager.register_event("*")
        async def on_event(message):
            print(f"Event: {message.event_type} - {message}")

        @manager.register_event("NewChannel")
        async def on_new_channel(message):
            print(f"New call: {message.get('Channel')}")

        @manager.register_event("Hangup")
        async def on_hangup(message):
            print(f"Call ended: {message.get('Channel')}")

        try:
            while True:
                await asyncio.sleep(1)
        except KeyboardInterrupt:
            print("Shutting down...")

asyncio.run(main())

Originate Calls

Originate multiple calls concurrently:

import asyncio
from asyncio_manager import Manager

async def originate_call(manager, number):
    try:
        response = await manager.originate(
            channel=f"SIP/provider/{number}",
            exten="100",
            context="from-internal",
            priority=1,
            caller_id="Test <1234>",
            timeout=30, async_=True,
        )
        return response.action_id if response.is_success else None
    except Exception as e:
        print(f"Error: {e}")
        return None

async def main():
    async with Manager(host="127.0.0.1", username="admin", secret="password") as m:
        await m.connect()
        numbers = ["5551234567", "5559876543"]
        tasks = [originate_call(m, n) for n in numbers]
        results = await asyncio.gather(*tasks)
        print(f"Successful: {sum(1 for r in results if r)}/{len(results)}")

asyncio.run(main())

FastAGI Server

Async IVR server. Uses only native async def (no deprecated asyncio.coroutine()):

import asyncio
from asyncio_manager import FastAGIServer, Request

async def ivr_handler(request: Request):
    await request.answer()
    await request.say_digits("1")
    await request.say_digits("2")

    result = await request.get_data("welcome", 5000, 2)

    if result == "1":
        await request.exec_("Goto", "sales,s,1")
    elif result == "2":
        await request.exec_("Goto", "support,s,1")
    else:
        await request.hangup()

async def main():
    server = FastAGIServer()
    server.add_script("menu", ivr_handler)
    await server.start(host="0.0.0.0", port=4574)

    try:
        while True:
            await asyncio.sleep(1)
    except KeyboardInterrupt:
        await server.stop()

asyncio.run(main())

CallManager

Full call lifecycle tracking:

import asyncio
from asyncio_manager import CallManager

async def main():
    mgr = CallManager(host="127.0.0.1", username="admin", secret="password")

    async with mgr:
        call = await mgr.originate("SIP/2000", "1000", context="default")
        print(f"Call originated: {call.id}")

        try:
            await call.wait_for_answer(timeout=30.0)
            print("Answered")
            await call.wait_for_hangup()
            print("Hung up")
        except asyncio.TimeoutError:
            print("Timeout — hanging up")
            await call.hangup()

asyncio.run(main())

EventList

For actions that return multiple events followed by *Complete (e.g. QueueStatus):

import asyncio
from asyncio_manager import Manager

async def main():
    async with Manager(host="127.0.0.1", username="admin", secret="password") as m:
        await m.connect()

        messages = await m.send_action_and_wait_all({
            "Action": "QueueStatus", "Queue": "support",
        })

        print(f"Received {len(messages)} messages:")
        for msg in messages:
            print(f"  {msg.event_type}: {dict(msg)}")

asyncio.run(main())

CLI

Send an action

asyncio-manager --username admin --secret password action Ping

Originate a call

asyncio-manager --username admin --secret password \
    originate SIP/100 200 --caller-id "Test <1234>"

Monitor events

asyncio-manager --username admin --secret password \
    monitor --filter "NewChannel"

Run Asterisk CLI command

asyncio-manager --username admin --secret password \
    command "pjsip show endpoints"

FAQ / Troubleshooting

ConnectionError: Cannot connect

Verify Asterisk is running (systemctl status asterisk) and port 5038 is accessible (netstat -tlnp | grep 5038).

AuthenticationError: Invalid credentials

Check /etc/asterisk/manager.conf and ensure the user has read/write permissions.

TimeoutError: Operation timed out

Increase timeouts in the Manager constructor:

Manager(timeout=15.0, read_timeout=60.0, connect_timeout=30.0)

How do I verify the library works without a real Asterisk?

Run the unit tests:

pip install -e ".[dev]"
python -m pytest tests/ -v
